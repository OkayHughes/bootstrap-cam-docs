Detailed Description

configuration file parser

Author
Magnus Hagdorn
Date
May 2004

procedures used to parse configuration files. The file syntax is similar to MS Windows style ini files or files that can be parsed using the Python configuration file parser module.

The file is split up into sections. Each section appears in [] brackets. Each section can contain a number of key, value pairs. Key, value pairs are separated by : or =.

Strings starting with any of the following characters are ignored (comments): !, # or ;

The sections are stored in a linked list. The key-value pairs of each section are also stored in linked lists. The module provides accessors to query the data structure.
