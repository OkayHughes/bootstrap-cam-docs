Detailed Description

The module 'gfdl_cloud_microphys' contains the full GFDL cloud microphysics (Chen and Lin 2013) [chen2013seasonal] and (Zhou et al. 2019) [zhou2019toward].

The module is paired with 'fv_cmp', which performs the "fast" processes author Shian-Jiann Lin, Linjiong Zhou
