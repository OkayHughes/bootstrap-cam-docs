Detailed Description

PIO async message handling. This file contains the code which runs on the IO nodes when async is in use. This code waits for messages from the computation nodes, and responds to messages by running the appropriate netCDF function.

Note that when calling the PIOc_* funtion, the return code should be ignored. It is handled within the function. Only errors in internal pio_msg code should return an error from the handler function.

Author
Ed Hartnett
