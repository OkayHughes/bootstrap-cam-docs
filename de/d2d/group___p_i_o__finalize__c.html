int PIOc_finalize ( int  iosysid)

Clean up internal data structures, and free MPI resources, associated with an IOSystem. This is the old name for PIOc_free_iosystem(). This function is maintained for backward compatibility. Use PIOc_free_iosystem() for new code.

Parameters
iosysidthe io system ID provided by PIOc_Init_Intracomm() or PIOc_init_async().
Returns
0 for success or non-zero for error.
