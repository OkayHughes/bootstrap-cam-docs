Test the decomp read/write functionality. Given an ioid for a 3D decomposition, this function will write a decomp file, then read it in to ensure the correct values are read.

Parameters
iosysidthe IO system ID.
ioidthe ID of the decomposition.
num_flavorsthe number of IOTYPES available in this build.
flavorarray of available iotypes.
my_rankrank of this task.
rearrangerthe rearranger to use (PIO_REARR_BOX or PIO_REARR_SUBSET).
test_commthe MPI communicator for this test.
dim_lenarray of length 3 with dim lengths.
expected_maplenpointer to array of length TARGET_NTASKS with the maplen we expect to get for each of the tasks running this test.
pio_typethe type we expect to be associated with this decomposition.
full_maplenthe length of the full map.
pointerto expected map, an array of TARGET_NTASKS * max_maplen.

0 for success, error code otherwise.
