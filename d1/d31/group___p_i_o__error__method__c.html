int PIOc_Set_File_Error_Handling ( int  ncid,
int  method 
)

Set the error handling method to be used for subsequent pio library calls, returns the previous method setting. Note that this changes error handling for the IO system that was used when this file was opened. Other files opened with the same IO system will also he affected by this call. This function is supported but deprecated. New code should use PIOc_set_iosystem_error_handling(). This method has no way to return an error, so any failure will result in MPI_Abort.

Parameters
ncidthe ncid of an open file
methodthe error handling method
Returns
old error handler
