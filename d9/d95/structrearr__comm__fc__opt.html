int rearr_comm_fc_opt::max_pend_req

Max pending requests (PIO_REARR_COMM_UNLIMITED_PEND_REQ => unlimited pend req). This is the number of messages allowed to be in flight at one time. On some systems posting all messages at once creates a significant bottleneck in communications and throttling in this manner improves overall performance.
